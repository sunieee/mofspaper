# p2t

用于将pdf格式的论文生成html（使用pdf2htmlEX），并一键转换成txt格式（convert.js）


## 安装
全过程一键安装：
```
cd /usr/bin
wget http://actvis.cn/data2/share/script/p2t.sh -O p2t && chmod 777 p2t
p2t <file1> <file2> ...
```

详细安装见：PDF转换相关过程.pdf
